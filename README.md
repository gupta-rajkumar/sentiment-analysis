# sentiment-analysis
A basic sentiment analyzer which takes tweets from the twitterAPI, analyzes it and and shows how many tweets were in favour of it and how many were not in favour of that particular keyword. 10 tweets are analyzed at a time. Textblob library was used for the purpose of analysis.

1. Copy your bearer token.
2. Open up the terminal and type the following command.
3. export BEARER_TOKEN={YOUR BEARER TOKEN}
